一个任务集有一个可行调度的必要条件是 U<=1
但是Up<=1时，RM算法未必是可行的
如果Up < Ulub, RM算法是可行的
如果Ulub < Up <= 1， 则不能确定RM算法是否可行
如何计算Ulub？ Ulub = n(2^1/n - 1)
如何判断一个任务集是否能用RM算法进行调度？
1、首先使用必要条件判断 Up <= 1
2、使用 Up <= Ulub 进行判断，RM算法的Ulub=n(n^1/n - 1)
3、使用 累乘(Ui + 1) <= 2 进行判断
4、使用类似DM算法那种方式去判断
